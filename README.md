# Online Retail Data Analysis Project

## Overview
This repository contains a detailed analysis of a transnational dataset which contains all the transactions occurring between 01/12/2010 and 09/12/2011 for a UK-based and registered non-store online retail.

## Dataset Features
- **Transactions:** 465,000+
- **Columns:** InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country.
- **Geographic Coverage:** Includes markets in the UK, Europe, Australia, and Asia.

## Key Objectives
1. **Data Cleaning:** Handling missing values (specifically in CustomerID) and removing outliers.
2. **Sales Trends:** Visualizing revenue growth over time and identifying peak sales periods.
3. **Geographic Analysis:** Comparing market performance between the UK and international regions.
4. **Product Analysis:** Identifying best-selling items and stock-keeping unit (SKU) performance.

## Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebooks
